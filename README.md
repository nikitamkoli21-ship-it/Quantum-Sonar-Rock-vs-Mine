# Quantum SONAR Rock vs Mine Prediction

This project predicts whether an underwater object is a **Rock** or a **Mine**
using a **Hybrid Quantum Machine Learning approach**.

## Technologies Used
- Python
- Qiskit
- Quantum Machine Learning
- Scikit-learn
- Google Colab

## Dataset
- SONAR Rock vs Mine dataset

## How to Run
1. Upload `sonar.csv` in Google Colab
2. Open `Quantum_SONAR.ipynb`
3. Run all cells from top to bottom

## Output
- Prediction: Rock / Mine
- Confidence score
